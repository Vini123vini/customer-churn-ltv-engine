# Customer Churn Prediction & Lifetime Value Engine

## Project Overview

This project predicts customer churn and estimates Customer Lifetime Value (LTV) using machine learning techniques. The goal is to help businesses identify customers likely to leave and prioritize retention strategies.

## Dataset

* Telco Customer Churn Dataset
* Source: IBM / Kaggle

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* XGBoost
* SHAP
* FastAPI
* PostgreSQL
* Streamlit

## Project Structure

* data/ : Raw and processed datasets
* notebooks/ : EDA and model development
* src/ : Source code
* models/ : Saved machine learning models
* reports/ : Project reports
* dashboard/ : Visualization dashboard
